<h1 align=center>FilePro</h1>

<h4 align=center>A small File hosting/uploading service made with Flask and SQL.</h4>


## Installation

```shell
pip install -r requierments.txt
```

## Start

Without Debugging:
```
python wsgi.py
```

With Debugging:
```
python -m filepro.main
```
